Research-grade peptide supply website for Elysian Bio LLC. Built as a single-page HTML application featuring a full product catalog with 14 research compounds, individual product detail pages with size variants, quantity selection, and COA (Certificate of Analysis) cards, a wholesale B2B inquiry section with tiered pricing, a COA library, and a multi-tab checkout supporting crypto, card, and wire payments. All products are marketed strictly for in-vitro laboratory research use only in compliance with FDA guidelines. To deploy, simply upload `index.html` to any static host such as GitHub Pages, Vercel, or Netlify.
